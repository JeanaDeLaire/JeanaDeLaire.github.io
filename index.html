Visit Rhodypedia
View this project on github

This full-stack app is deployed to Heroku and uses Rails as its back-end framework. The front-end utilizes HTML5, CSS and Bootstrap. Javascript and jQuery handle site interaction and help present data to users. Handlbars displays the information from the database through a grid template. The searchbar filters through keywords through javascript and pushes in relevant matches into a new array which is passed through handlebars. This project was completed for a full-stack development training course and I intend to further build out the site and add in the ability for users to add photos, comments, and site moderators.

Visit FileBucket
View this project on github

This full-stack app was constructed for a team project during a web development immersive program through General Assembly. As a file management system with a focus on image files, FileBucket is deployed to Heroku and uses Express and mongodb to build the api. Image files are stored to AWS. Only user owned resources can be updated and deleted, while all resources within the database are open view. The front-end utilizes HTML5, CSS, Bootstrap, Javascript, jQuery, and a responsive Handlebars grid template for the image gallery.
